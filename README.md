# esp32-chip-info
Identify ESP32 chip by software.

### Example output to Serial
```
ESP32 model: ESP32
Chip Model: ESP32-D0WD-V3
Chip ID: 2B32A7B0
Chip Revision: 3
Core: 2
Clock Speed: 240MHz
Features:
- WIFI BGN
- BLUETOOTH
- BLUETOOTH LE
- EXTERNAL FLASH MEMORY
Flash Memory Size: 4.194304 MB, 4 MiB, 4194304 bytes
```
